🔄 End-to-End Data Flow

1) Mobile Upload User captures photo and submits survey payload from mobile UI.
2) Backend Intake Photo + metadata received by survey routes and stored.
3) Cloud Inference Backend proxy sends image bytes to Roboflow hosted model.
4) Normalize & Persist Predictions are adapted and written to DB records.
5) Website Visuals Frontend reads processed results and displays project insights.
